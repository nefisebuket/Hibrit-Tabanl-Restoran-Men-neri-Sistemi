Hibrit Arama ve RAG Tabanlı Restoran Menü Bilgi Erişim Sistemi

BM25 + Sentence-BERT + RRF hibrit arama, alerjen filtreleme ve Llama 3.2 (Ollama) destekli RAG chatbot içeren restoran menü öneri sistemi. 160K+ ürün, 5 şehir, ASP.NET Core web arayüzü + Flask API.

Özellikler:
- Hibrit bilgi erişimi: BM25 + anlamsal arama (all-MiniLM-L6-v2) + RRF
- 10 alerjen grubu için filtreleme ve uyarı sistemi
- RAG mimarisi ile Llama 3.2 üzerinden doğal dil chatbot
- Şehir bazlı filtreleme (New York, Chicago, San Francisco, Boston, Austin)
- Tamamen yerel çalışma — bulut bağımlılığı yok

Teknolojiler: Python, Flask, ASP.NET Core, Ollama, Hugging Face
